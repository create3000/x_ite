default_cylinder.x3d

Test default CylinderSensor properties. A default CylinderSensor is assigned to an IndexedFaceSet geometry. The rotation_changed fields from the CylinderSensor are routed to the transform node for the same geometry. As a result, moving the geometry should result in, when moved from the side, a "cylinderlike" rotation when viewed from the front, back or side. The geometry should keep its orientation after each "drag" motion of the pointing device.
